# GB50217 电缆经济电流截面计算器 V5

## 技能描述

基于 GB 50217-2018《电力工程电缆设计标准》附录B，实现电力电缆经济电流截面的自动计算与选型推荐。

## V5 版本更新

### 核心改进：经济电流密度公式化计算

V5版本将经济电流密度从"查表法"改为"GB50217附录B公式动态计算"：

```
Jec = √(1000 × a / (3 × ρ × τ × C2 × N))
```

| 参数 | 含义 | 来源 |
|------|------|------|
| a | 电缆成本系数 (元/mm²·m) | 铜价/铝价实时计算 |
| ρ | 导体电阻率 (Ω·mm²/m) | 铜0.0172 / 铝0.0283 |
| τ | 最大损耗小时数 (h) | 由Tmax估算 |
| C2 | 电价 (元/kWh) | 用户输入 |
| N | 年金现值系数 | 贴现率、经济寿命 |

### 验算结果

| 工况 | 公式Jec | 公式截面 | LCC最优 | 差异 |
|------|---------|---------|---------|------|
| 铜芯200A | 1.297 | 150mm² | 150mm² | **0档** |
| 铜芯300A高电价 | 1.002 | 300mm² | 300mm² | **0档** |
| 铝芯200A | 0.316 | 630mm² | 630mm² | **0档** |
| 铜芯150A长距离 | 1.396 | 95mm² | 95mm² | **0档** |

**结论：经济截面与LCC最优截面完全一致！**

## 核心功能

1. **经济电流密度计算**
   - 按GB50217附录B公式动态计算Jec
   - 根据实际电价、铜价、贴现率、经济寿命自动调整
   - 无需查表，适应不同经济环境

2. **经济截面选择**
   - 自动推荐最经济的电缆标称截面
   - 全生命周期成本(LCC)分析验证
   - 考虑技术条件与经济条件的综合平衡

3. **多因素校核**
   - 热稳定校验（GB50217-2018）
   - 电压降校验（配电设计手册第四版）
   - 持续载流量校验（GB50217-2018附录C）

4. **电缆型号支持**
   - 铜芯：YJY22（XLPE绝缘钢带铠装PE护套）
   - 铝芯：YJLHY22（XLPE绝缘铝合金导体钢带铠装PE护套）
   - 电压等级：1kV（4芯）、10kV（3芯）

- **贴现率**: 默认 6%

## 计算原理

### 经济电流密度公式 (GB50217 附录 B.0.1)

**LCC总费用公式：**
```
LCC = a × S × L + (3 × I² × ρ × L × τ × C2 / (1000 × S)) × N
```

**对S求导，令dLCC/dS = 0：**
```
S = I × √(3 × ρ × τ × C2 × N / (1000 × a))
Jec = I / S = √(1000 × a / (3 × ρ × τ × C2 × N))
```

### 选型逻辑

```
经济截面 = Imax / Jec（Jec按GB公式动态计算）
最小要求截面 = max(热稳定截面, 载流量截面, 电压降截面)

若经济截面 ≥ 最小要求截面：
    按经济选型（从最接近经济截面的规格开始LCC对比）
若经济截面 < 最小要求截面：
    按最小要求截面选型（确保技术安全）
```

### 热稳定校验 (GB50217-2018)

```
S_min = (I_k × √t) / C

其中：
- I_k: 短路电流 (A)
- t: 短路持续时间 (s)
- C: 热稳定系数 (铜=137, 铝=93)
```

### 载流量计算 (GB50217-2018 附录C)

```
I = I_ref × K_t × K_install × K_soil

基准条件：土壤直埋，25℃，热阻系数2.0 K·m/W，XLPE绝缘(90℃)
铜芯载流量 = 铝芯 × 1.29
```

## 输出结果

1. **推荐截面**: 最经济的标称截面 (mm²)
2. **经济电流密度**: 公式计算的Jec值 (A/mm²)
3. **技术校核结果**:
   - 热稳定校验: 通过/不通过
   - 电压降校验: 通过/不通过 (实际压降值)
   - 载流量校验: 通过/不通过
4. **经济对比表**: 多个候选截面的LCC对比
5. **初始投资明细**: 电缆、通道、附件、其他费用
6. **全生命周期成本**: 损耗费现值、运维费现值、总现值

## 注意事项

1. 本计算器基于 GB 50217-2018 标准
2. 计算结果仅供参考，实际工程设计需结合具体工况
3. 备用回路电缆不宜按经济电流密度选择截面
4. 当经济截面小于技术条件要求截面时，应按技术条件选择
