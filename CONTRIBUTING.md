# 贡献指南

感谢你对 GB50217 电缆经济选型计算器项目的关注！我们欢迎任何形式的贡献。

## 🤝 如何贡献

### 报告 Bug

如果你发现了 bug，请：

1. 先搜索 [Issues](https://github.com/lh10kv-gif/gb50217-calculator/issues) 确认是否已被报告
2. 使用 Bug 报告模板创建新 Issue
3. 提供详细的复现步骤和相关信息

### 提出新功能

如果你有新的功能建议：

1. 先搜索 [Issues](https://github.com/lh10kv-gif/gb50217-calculator/issues) 确认是否已被提出
2. 使用功能请求模板创建新 Issue
3. 详细描述功能需求和预期行为

### 提交代码

如果你想贡献代码：

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request

## 📋 代码规范

### Python 代码风格

- 遵循 [PEP 8](https://pep8.org/) 代码风格
- 使用有意义的变量和函数名
- 添加必要的注释和文档字符串

### 提交信息

提交信息应清晰描述更改内容：

```
<type>: <subject>

<body>

<footer>
```

**Type 类型：**
- `feat`: 新功能
- `fix`: 修复 bug
- `docs`: 文档更改
- `style`: 代码格式更改（不影响功能）
- `refactor`: 重构代码
- `test`: 添加或修改测试
- `chore`: 其他杂项

**示例：**
```
feat: 添加铝芯电缆载流量计算

- 更新载流量计算模块
- 添加铝芯电阻率参数
- 修正电压降计算公式

Closes #123
```

## 🧪 测试

在提交 PR 之前，请确保：

1. 代码能正常运行
2. 已测试新功能或修复的 bug
3. 没有引入新的 bug

## 📝 文档

如果更改了功能，请相应更新：

- README.md
- 函数/类的文档字符串
- 相关的使用说明

## 💬 沟通

在提交 PR 之前，建议：

1. 先创建 Issue 讨论你的想法
2. 在 PR 中引用相关的 Issue
3. 及时回复代码审查的评论

通过贡献代码，你同意你的贡献将根据 [MIT License](LICENSE) 进行许可。
