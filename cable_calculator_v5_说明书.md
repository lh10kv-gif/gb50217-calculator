## 1. 工具概述

### 1.1 工具定位

本工具是一款基于 **GB 50217-2018《电力工程电缆设计标准》** 的电缆经济选型计算软件，用于电力工程设计中电缆截面的技术校验与经济优化选型。

### 1.2 适用范围

- 电压等级：1kV（低压）、10kV（中压）
- 导体材料：铜芯、铝芯
- 敷设方式：直埋、穿管、桥架、电缆沟
- 应用场景：电力工程设计、配电系统优化、工程经济性分析

### 1.3 核心特点

| 特点 | 说明 |
|------|------|
| **经济电流密度动态计算** | 按GB50217附录B公式，根据实际电价、铜价、贴现率动态计算Jec，而非查表 |
| **全生命周期成本分析** | 综合考虑初始投资、运行损耗、运维费用，计算30年总费用现值 |
| **多约束技术校验** | 热稳定、载流量、电压降三项技术条件同时校验 |
| **价格实时联动** | 电缆价格根据铜/铝原材料价格实时计算 |

### 1.4 版本历史

| 版本 | 主要更新 |
|------|----------|
| V5 | 经济电流密度改为GB50217附录B公式动态计算；修复候选截面筛选逻辑 |
| V4 | 增加土壤热阻系数选择 |
| V3 | 修正年损耗费用计算公式 |
| V2 | 增加全生命周期成本(LCC)分析 |
| V1 | 基础版本，查表法获取经济电流密度 |

## 3. 参数说明与原理

### 3.1 电缆基础参数

#### 3.1.1 电压等级

| 选项 | 说明 | 对应电缆 |
|------|------|----------|
| **10kV** | 中压配电 | YJY22-10kV-3芯（铜）/ YJLHY22-10kV-3芯（铝） |
| **1kV** | 低压配电 | YJY22-1kV-4芯（铜）/ YJLHY22-1kV-4芯（铝） |

**原理说明：**
- 10kV电缆采用3芯结构（三相对称）
- 1kV电缆采用4芯结构（三相+零线）
- 不同电压等级对应不同的载流量数据表

#### 3.1.2 导体材料

| 材料 | 电阻率 (Ω·mm²/m) | 密度 (g/cm³) | 热稳定系数 C |
|------|-------------------|--------------|--------------|
| **铜芯** | 0.017241 | 8.96 | 137 |
| **铝芯** | 0.028264 | 2.70 | 93 |

**原理说明：**
- 铜的导电率约为铝的1.64倍，同截面铜芯载流量约为铝芯的1.29倍
- 铜的热稳定系数更高，同短路条件下所需最小截面更小
- 铝芯价格较低，适合大截面、低负荷场景

#### 3.1.3 最大持续工作电流 (Imax)

**定义：** 电缆在正常运行时持续承载的最大负荷电流（A）

**获取方式：**
- 根据负荷计算确定
- 考虑负荷同时系数、需要系数等
- 取正常运行条件下的最大值（非短路电流）

**对选型的影响：**
- 直接决定载流量校验的最小截面要求
- 影响经济电流密度计算
- 影响年损耗费用计算

#### 3.1.4 年最大负荷利用小时数 (Tmax)

**定义：** 年用电量与最大负荷的比值，反映负荷的时间特性

```
Tmax = 年用电量 / 最大负荷 (h)
```

**典型值：**

| 用户类型 | Tmax 典型值 |
|----------|-------------|
| 单班制工厂 | 1500-2500 h |
| 两班制工厂 | 3000-4500 h |
| 三班制工厂 | 5000-7000 h |
| 商业建筑 | 2500-3500 h |
| 居民住宅 | 2000-3000 h |

**对选型的影响：**
- Tmax 越大，年损耗越大，经济电流密度越小，推荐截面越大
- 用于估算最大损耗小时数 τ

#### 3.1.5 功率因数 (cosφ)

**定义：** 有功功率与视在功率的比值

**典型值：**
- 工业负荷：0.85-0.90
- 商业负荷：0.90-0.95
- 民用负荷：0.90-0.92

**对选型的影响：**
- 影响电压降计算（无功功率在电抗上的压降）
- 影响最大损耗小时数 τ 的估算

#### 3.1.6 电缆长度 (L)

**单位：** 米 (m)

**对选型的影响：**
- 长度越大，电压降越大，可能需要增大截面
- 长度越大，初始投资越高
- 长度越大，年损耗越大

#### 3.1.7 允许电压降

**定义：** 电缆末端电压相对于首端的允许降低百分比

**标准推荐值：**

| 场景 | 允许电压降 |
|------|------------|
| 一般低压配电 | ≤5% |
| 照明回路 | ≤3% |
| 电动机回路 | ≤5%（频繁启动时≤3%） |
| 10kV配电 | ≤5% |

**原理说明：**
电压降由电阻压降和电抗压降两部分组成：
```
ΔU% = √3 × I × (R×cosφ + X×sinφ) × L / U × 100%
```

### 3.2 环境与短路参数

#### 3.2.1 敷设方式

| 方式 | 载流量修正系数 | 通道费用 (元/m) | 适用场景 |
|------|----------------|-----------------|----------|
| **直埋** | 1.00 | 150 | 一般场地，开挖方便 |
| **穿管** | 0.85 | 300 | 穿越道路、建筑物 |
| **桥架** | 1.10 | 100 | 室内、电缆沟上方 |
| **预留电缆沟** | 0.95 | 40 | 有现成电缆沟 |

**原理说明：**
- 穿管敷设散热条件差，载流量降低约15%
- 桥架敷设空气流通好，载流量提高约10%
- 敷设方式同时影响通道建设费用

#### 3.2.2 环境温度

**定义：** 电缆敷设环境的工作温度

**标准参考值：**
- 土壤直埋：25℃（GB50217附录C基准条件）
- 空气中敷设：40℃
- 电缆沟内：按实际测量

**对载流量的影响：**
```
载流量修正系数 K_t = √[(θ_m - θ) / (θ_m - θ_ref)]

其中：
- θ_m = 90℃（XLPE绝缘最高允许温度）
- θ = 实际环境温度
- θ_ref = 25℃（参考温度）
```

**示例：**
- 环境温度40℃时，K_t = √[(90-40)/(90-25)] = 0.877（载流量降低约12%）
- 环境温度15℃时，K_t = √[(90-15)/(90-25)] = 1.074（载流量提高约7%）

#### 3.2.3 土壤热阻系数

**定义：** 土壤对热量传递的阻碍程度，单位 K·m/W

**典型值与对应土壤类型：**

| 热阻系数 | 土壤类型 | 载流量修正系数 |
|----------|----------|----------------|
| 0.8 | 湿润土壤（含水量高） | 1.22 |
| 1.0 | 潮湿土壤 | 1.15 |
| 1.5 | 一般土壤 | 1.04 |
| **2.0** | **GB50217标准条件** | **1.00** |
| 2.5 | 较干土壤 | 0.94 |
| 3.0 | 干燥沙土、岩石 | 0.88 |

**原理说明：**
- 热阻系数越低，土壤散热越好，电缆载流量越高
- GB50217附录C载流量表的基准条件为2.0 K·m/W
- 实际工程应根据地质勘察报告确定

#### 3.2.4 短路电流 (Ik)

**定义：** 短路故障时通过电缆的最大短路电流有效值

**单位：** kA（千安）

**获取方式：**
- 由系统短路计算确定
- 查询上级变电站短路容量
- 典型值：10kV系统约20-31.5kA，低压系统约25-50kA

**对选型的影响：**
用于热稳定校验，计算最小热稳定截面：
```
S_min = (I_k × √t) / C

其中：
- I_k：短路电流 (A)
- t：短路持续时间 (s)
- C：热稳定系数（铜=137，铝=93）
```

#### 3.2.5 短路持续时间 (t)

**定义：** 从短路发生到保护动作切除的时间

**典型值：**
- 快速保护：0.1s
- 一般保护：0.2-0.5s
- 后备保护：1.0-2.0s

**对选型的影响：**
时间越长，热效应越大，所需最小截面越大

**单位：** 元/吨

**用途：** 实时计算电缆价格

**电缆价格计算公式：**
```
电缆价格 = (截面 × 芯数 × 导体密度 / 1000) × (单价/1000) / 线芯占比

示例：10kV铜芯150mm²电缆价格计算
= (150 × 3 × 8.96 / 1000) × (100000/1000) / 0.85
= 4.032 kg/m × 100 元/kg / 0.85
= 474.4 元/m
```

**线芯占比系数：**

| 材料 | 电压等级 | 线芯占比 |

| 铜 | 1kV | 90% |
| 铝 | 10kV | 70% |

### 3.4 经济参数

#### 3.4.1 经济寿命

**定义：** 电缆经济分析的周期

**典型值：** 30年

**用途：** 计算全生命周期成本时，考虑该年限内的损耗费用和运维费用现值

#### 3.4.2 贴现率

**定义：** 将未来费用折算为现值的利率

**典型值：** 6%（电力行业常用）

**用途：** 计算年金现值系数：
```
N = [1 - (1+r)^(-n)] / r

其中：
- r：贴现率（如6%则为0.06）
- n：经济寿命年数

示例：贴现率6%，经济寿命30年
N = [1 - 1.06^(-30)] / 0.06 = 13.76
```

#### 3.4.3 电价

**单位：** 元/kWh

**典型值：**
- 工业用电：0.6-0.9 元/kWh
- 商业用电：0.8-1.2 元/kWh
- 居民用电：0.5-0.6 元/kWh

**用途：** 计算年损耗费用

#### 3.4.4 运维费用

**单位：** 元/m·年

**定义：** 电缆每年的运行维护费用

**典型值：** 5 元/m·年

**包含内容：**
- 定期巡检费用
- 预防性试验费用
- 维修费用分摊

### 4.2 热稳定校验

**目的：** 确保电缆能承受短路电流的热效应，不发生绝缘损坏

**计算公式：**
```
S_min = (I_k × √t) / C

其中：
- S_min：最小热稳定截面 (mm²)
- I_k：短路电流有效值 (A)
- t：短路持续时间 (s)
- C：热稳定系数
  - 铜芯 XLPE绝缘：C = 137
  - 铝芯 XLPE绝缘：C = 93
```

**校验标准：**
所选截面 S ≥ S_min，则热稳定校验通过

**示例：**
```
已知：I_k = 25kA，t = 0.1s，铜芯电缆

计算：
S_min = (25000 × √0.1) / 137
      = (25000 × 0.316) / 137
      = 57.7 mm²

结论：选择 ≥ 70mm² 的截面即可满足热稳定要求
```

### 4.3 载流量校验

**目的：** 确保电缆在正常运行时不发生过热

**计算步骤：**

**Step 1：获取基准载流量**

根据GB50217-2018附录C查表获取：

| 电压 | 材料 | 截面(mm²) | 基准载流量(A) |
|------|------|-----------|---------------|
| 10kV | 铝芯 | 150 | 219 |
| 10kV | 铜芯 | 150 | 283 (=219×1.29) |
| 1kV | 铝芯 | 150 | 235 (=247×0.95) |
| 1kV | 铜芯 | 150 | 303 |

**Step 2：计算修正载流量**

```
I = I_ref × K_t × K_install × K_soil

其中：
- I_ref：基准载流量
- K_t：温度修正系数 = √[(90-θ)/(90-25)]
- K_install：敷设方式修正系数
- K_soil：土壤热阻修正系数
```

**Step 3：校验**

```
I ≥ Imax → 通过
I < Imax → 需增大截面
```

**示例：**
```
已知：铜芯10kV电缆，150mm²，环境温度35℃，穿管敷设，土壤热阻2.0

计算：
K_t = √[(90-35)/(90-25)] = √(55/65) = 0.92
K_install = 0.85（穿管）
K_soil = 1.0

I = 283 × 0.92 × 0.85 × 1.0 = 221 A

若 Imax = 200A，则 221A > 200A，校验通过
```

### 4.4 电压降校验

**目的：** 确保电缆末端电压满足设备运行要求

**计算公式：**
```
ΔU% = √3 × I × (R×cosφ + X×sinφ) × L / U × 100%

其中：
- I：负荷电流 (A)
- R：单位长度电阻 (Ω/km)
- X：单位长度电抗 (Ω/km)
- L：电缆长度 (km)
- U：系统电压 (V)
- cosφ：功率因数
```

**电阻计算：**
```
R = K / S (Ω/km)

其中：
- 铜芯：K = 18.5 (10kV) 或 20.0 (1kV)
- 铝芯：K = 30.3 (10kV) 或 32.8 (1kV)
- S：截面 (mm²)
```

**电抗取值：**
- 10kV电缆：X = 0.09 Ω/km
- 1kV电缆：X = 0.08 Ω/km

**示例：**
```
已知：铜芯10kV电缆，150mm²，Imax=200A，L=500m，cosφ=0.9

计算：
R = 18.5/150 = 0.123 Ω/km
X = 0.09 Ω/km
sinφ = √(1-0.9²) = 0.436

ΔU% = √3 × 200 × (0.123×0.9 + 0.09×0.436) × 0.5 / 10000 × 100%
    = 1.732 × 200 × (0.111 + 0.039) × 0.5 / 10000 × 100%
    = 0.26%

结论：0.26% < 5%，电压降校验通过
```

### 4.5 经济电流密度计算（V5核心）

**GB50217附录B公式推导：**

**LCC总费用函数：**
```
LCC = a×S×L + (3×I²×ρ×L×τ×C2 / (1000×S)) × N

其中：
- a：电缆单位截面成本 (元/mm²·m)
- S：截面 (mm²)
- L：长度 (m)
- I：电流 (A)
- ρ：电阻率 (Ω·mm²/m)
- τ：最大损耗小时数 (h)
- C2：电价 (元/kWh)
- N：年金现值系数
```

**对S求导，令 dLCC/dS = 0：**
```
dLCC/dS = a×L - (3×I²×ρ×L×τ×C2×N) / (1000×S²) = 0

解得：
S = I × √(3×ρ×τ×C2×N / (1000×a))
```

**经济电流密度：**
```
Jec = I/S = √(1000×a / (3×ρ×τ×C2×N))
```

**参数计算：**

| 参数 | 计算方式 |
|------|----------|
| a (电缆成本系数) | a = 芯数 × 密度 / 1000 × 单价(元/kg) / 线芯占比 |
| ρ (电阻率) | 铜=0.017241，铝=0.028264 Ω·mm²/m |
| τ (损耗小时) | 根据Tmax估算 |
| C2 (电价) | 用户输入 |
| N (年金现值系数) | N = [1-(1+r)^(-n)]/r |

**示例：**
```
已知：铜芯10kV，Tmax=5000h，铜价100000元/吨，电价0.4元/kWh，贴现率6%，寿命30年

计算：
a = 3 × 8.96/1000 × 100/0.85 = 3.16 元/mm²·m
ρ = 0.017241 Ω·mm²/m
τ ≈ 3400 h（由Tmax估算）
C2 = 0.4 元/kWh
N = 13.76

Jec = √(1000×3.16 / (3×0.017241×3400×0.4×13.76))
    = √(3160 / 967)
    = √3.27
    = 1.81 A/mm²

若 Imax = 200A：
S_eco = 200 / 1.81 = 110.5 mm² → 选择 120mm²
```

### 4.6 全生命周期成本对比

**计算公式：**
```
LCC = 初始投资 + 年损耗费用 × N + 年运维费用 × N

初始投资 = 电缆成本 + 通道费用 + 附件费用 + 其他费用

年损耗费用 = 3 × I² × R × L / 1000 × τ × 电价

年运维费用 = 运维费率 × L
```

**对比方法：**
- 对多个候选截面计算LCC
- 选择LCC最小的截面作为推荐方案

## 7. 注意事项与限制

### 7.1 使用限制

1. **电压等级限制**
   - 仅适用于1kV和10kV电压等级
   - 35kV及以上电压等级需参考其他标准

2. **电缆类型限制**
   - 仅适用于XLPE绝缘（交联聚乙烯）电缆
   - 不适用于PVC绝缘、油浸纸绝缘等

3. **敷设场景限制**
   - 载流量数据基于土壤直埋敷设
   - 空气中敷设需选用相应载流量表
   - 特殊环境（水下、高温区等）需单独考虑

4. **经济分析限制**
   - 未考虑通货膨胀
   - 未考虑电价上涨趋势
   - 未考虑电缆残值

### 7.2 注意事项

1. **参数准确性**
   - 输入参数应基于实际工程条件
   - 短路电流应由系统计算确定
   - 环境温度应考虑极端工况

2. **结果验证**
   - 计算结果仅供参考
   - 重要工程应进行详细计算复核
   - 最终选型需符合设计规范

3. **价格更新**
   - 铜/铝价格应按当前市场行情更新
   - 通道费用应按当地造价信息调整

4. **特殊情况处理**
   - 备用电缆不宜按经济电流密度选型
   - 消防电缆应满足防火要求
   - 腐蚀环境需考虑防腐措施

### 8.4 常见问题

**Q1：为什么推荐截面比经济截面大？**

A：推荐截面必须同时满足三项技术条件（热稳定、载流量、电压降）。当经济截面小于技术要求的最小截面时，系统会自动选择满足技术条件的最小截面。

**Q2：如何理解全生命周期成本？**

A：全生命周期成本(LCC) = 初始投资 + 运行期间的损耗费用现值 + 运维费用现值。LCC最小表示在整个使用周期内总费用最低，是最经济的选择。

**Q3：经济电流密度为什么是动态计算的？**

A：传统的查表法给出的经济电流密度是固定值，无法反映实际电价、材料价格、贴现率等因素的变化。V5版本采用GB50217附录B公式，根据实际参数动态计算，结果更准确。

**Q4：土壤热阻系数如何确定？**

A：应根据地质勘察报告确定。无勘察数据时，一般土壤可取2.0 K·m/W（GB标准条件），干燥沙土取3.0，潮湿土壤取1.0-1.5。
